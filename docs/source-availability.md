# Source Availability

The full implementation source is private.

## Why the Source Is Not Public

The original implementation was produced for university coursework. Publishing the source tree, testbenches, notebooks, build scripts, or bitstream-generation flow could expose restricted assessment material or help future students complete a similar assignment.

For that reason, this repository is a public technical dossier rather than a source-code release.

## Public in This Repository

- Architecture summary.
- FPGA hardware summary.
- Processing-system summary.
- Testing and validation summary.
- Project contribution statement.
- Non-sensitive diagrams and plots.

## Not Published

- SystemVerilog RTL.
- Python processing-system source.
- C++ CPU baseline source.
- Jupyter bring-up notebooks.
- Testbenches.
- Vivado build scripts.
- Generated IP/block-design artefacts.
- Bitstreams and hardware handoff files.
- Controller firmware.

## Interview Discussion

I can discuss the design, implementation approach, debugging process, timing/resource tradeoffs, and PS/PL integration. Where permitted by university policy, I can provide supervised evidence of code ownership without making restricted coursework material public.
